## ✨ Features

- **Advanced Moderation**: Powerful tools for managing your server including timeouts, warnings, and comprehensive mod logs.
- **Image Manipulation & Fun**: Generates custom welcome/goodbye cards, "wasted" overlays, Trump tweets, and more using Canvas.
- **Anime Integration**: Search and track your favorite anime using [Kitsu.js](https://kitsu.io/).
- **Lightweight Database**: Powered by Quick.db for fast and reliable data storage (JSON/SQLite).
- **Robust Command Handler**: Dynamically loads commands and categories with built-in cooldowns.

## 🚀 Getting Started

### Prerequisites

Ensure you have the following installed on your machine:
- [Node.js](https://nodejs.org/) (v16.x or newer recommended)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Anish-530/Suzushi.git
cd suzushi
```

2. Install the necessary dependencies:
```bash
npm install
```

3. Configure your environment:
Create a `.env` file in the root directory and add your bot token:
```env
token=YOUR_DISCORD_BOT_TOKEN_HERE
```

### Usage

To start Suzushi, run the following command:
```bash
node index.js
```
*Alternatively, use a process manager like \`pm2\` to keep the bot alive 24/7.*
